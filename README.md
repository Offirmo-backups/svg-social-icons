# svg-social-icons

## Abstract

SVG social icons based (mostly) on Squarespace icons.

## Install

`npm install svg-social-icons --save`

## Usage
`icon(<network>)

```js
var icons = require('svg-social-icons');

// return facebook icon 
icons('facebook');
```
